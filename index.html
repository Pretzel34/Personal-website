Experience

Software Engineeer
Mutual of Enumclaw Jan 2023 Present

At Mutual of Enumclaw, I modernized legacy insurance applications by transitioning from C# and .aspx to modern JavaScript frameworks like React, Next.js, and Vue.js. I focused on improving performance, consolidating codebases, and implementing automated testing with Playwright. I also mentored interns, integrated logic to meet state regulatory requirements, and built critical tools to support underwriting and quoting processes, helping streamline operations and enhance the user experience.

Software Engineer
Trader PhD Nov 2021 - Jan 2023

At Trader PhD, I led the transition from outsourced development to an in-house engineering team, establishing DevOps procedures and documenting local setup processes. I managed the rebuild of core applications using React Native, modified APIs in Node.js, and monitored AWS and production environments. I also coordinated with external UI/UX and DevOps teams, migrated version control from SVN to Git, and rebuilt the main website with a focus on SEO and analytics.

Software Engineer
LogicL LLC Jan 2019 - May 2021

At LogicL LLC, I served as the lead developer for mobile framework projects using React Native, NativeScript, and Flutter. I guided client projects through the full software development lifecycle, managed high-impact systems like Young Living’s commission program via SSH servers, and improved SEO practices across internal and client websites. I also developed and maintained a wide range of web applications using Angular, React, JavaScript, and Hugo, while designing backend architectures for high-end retail sites.
